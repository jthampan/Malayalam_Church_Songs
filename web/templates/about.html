About Church Songs PPT Generator

Purpose

This web application generates PowerPoint presentations for Holy Communion Services in Malayalam and English. It automatically searches for hymns in the church's hymn book and creates professionally formatted slides.

Features

  • ✅ Automatic hymn search from KK hymnal and service archives
  • ✅ Generates summary slide with all hymns
  • ✅ Holy Communion intro slides with hymn titles
  • ✅ Removes headers and decorative elements from KK slides
  • ✅ Supports batch file upload or manual entry
  • ✅ Customizable service date

How to Use

  1. Enter Service Date: Optional, in format "DD Month YYYY"
  2. Choose Input Method:
    • Upload a .txt batch file with your song list
    • OR manually type songs in the text area
  3. Click Generate: The PowerPoint will be created and downloaded automatically

Batch File Format

Each line should contain: HymnNumber|Label|OptionalTitle

91|Opening|
123|Offertory|
313|Communion|
456|Closing|

Supported Labels

  • Opening - Opening Hymn
  • ThanksGiving - Thanksgiving Prayers (displays as "B/A")
  • Offertory - Offertory Hymn
  • Message - Message Slide (no hymn number)
  • Confession - Confession Hymn
  • Communion - Holy Communion Hymn
  • Closing - Closing Hymn
  • Dedication - Dedication Hymn

Technical Details

This application searches through the church's PowerPoint archives to find the requested hymns. It prioritizes recent service presentations and falls back to the KK hymnal when needed.

All hymn titles are automatically extracted and displayed in the summary slide and Holy Communion intro slides.
